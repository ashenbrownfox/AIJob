## Prompt #1: Persona Builder

<div style="font-size: 0.65em;">

```markdown
I'm preparing for a career fair and need to build a comprehensive 
profile of my skills, experience, and goals. I'll provide my resume, 
but I also want to add context that isn't explicitly written.

[PASTE YOUR RESUME HERE]

Additional context:
- Courses I've excelled in: [LIST]
- Projects I'm proud of: [DESCRIBE]
- Technical skills I'm developing: [LIST]
- Soft skills I bring: [DESCRIBE]
- Career goals (6 months - 2 years): [DESCRIBE]
- Industries I'm interested in: [LIST]
- Work environment preferences: [DESCRIBE]

Please create a comprehensive persona profile that captures my 
strengths, skills, and career aspirations. Format this as a 
structured profile I can reference.
```

## Prompt #2: Company Analysis

<div style="font-size: 0.7em;">

```markdown
I found this company in my university's career fair catalog:

[PASTE COMPANY DESCRIPTION/LISTING]

Based on this information, please extract and organize:
1. What roles/positions are they recruiting for?
2. What specific technical skills are they seeking?
3. What soft skills or qualities do they emphasize?
4. Is this a paid opportunity? (internship/full-time)
5. Duration (if internship - 3 months? 6 months? 12 months?)
6. Location(s) of work
7. Any special requirements (GPA, graduation date, etc.)

Format this as a structured company profile.
```

</div>

---

## Prompt #3: Skills Match Analysis

<div style="font-size: 0.75em;">

```markdown
Based on my persona profile from earlier and this company profile, 
analyze:

1. Match Score (0-10): How well do my skills align?
2. Strengths: Which of my skills are most relevant?
3. Gaps: What am I missing (if anything)?
4. Bridge Strategy: How can I position my experience to fill gaps?
5. Talking Points: 3-5 specific things I should mention when 
   talking to their recruiter

Be honest about the match quality.
```

</div>

---

## Prompt #4: Build the Knowledge Table

<div style="font-size: 0.7em;">

```markdown
Add this company to a comparison table with these columns:
- Company Name
- Role Type
- Match Score (0-10)
- Top 3 Relevant Skills I Have
- Compensation (Paid/Unpaid/Unknown)
- Duration
- Key Talking Points
- Priority Level (High/Medium/Low)

After analyzing multiple companies, show me the complete table 
sorted by Priority Level and Match Score.
```

## Prompt #5: Career Fair Strategy

<div style="font-size: 0.65em;">

```markdown
Based on the company comparison table, create my career fair 
action plan:

1. TOP PRIORITY (90+ match): Companies I MUST visit
   - Estimated time per company: 20-30 minutes
   - Best time to visit: [early/mid/late in fair]
   - Total time allocation

2. MEDIUM PRIORITY (70-89 match): Strong backup options
   - Time per company: 15-20 minutes
   - Visit if time permits

3. LOW PRIORITY (<70 match): Skip or very brief visit
   - Only if I have extra time

4. WILD CARDS: Companies slightly outside my current skillset 
   but with strong interest

Include a suggested schedule/route through the fair.
```

</div>

---

## Prompt #6: Conversation Prep

<div style="font-size: 0.7em;">

```markdown
For [COMPANY NAME], create a 30-second elevator pitch I can use 
when approaching their booth. Include:

1. Strong opening hook (reference specific company need/value)
2. My relevant experience (2-3 key points from my persona)
3. Specific question about their role/opportunity
4. Confident closing

Make it conversational, authentic, and memorable. Avoid generic 
phrases like "I'm passionate about" or "I'm a hard worker."
```

</div>
